Mi trabajo

Explora mis proyectos destacados para obtener una visión más detallada de mi trabajo y cómo contribuyo al mundo del diseño y desarrollo web.

CONTRALUZ

Mi proyecto más reciente, "Contraluz", es un sitio web desarrollado desde cero en la plataforma Shopify. Fui la coordinadora de este proyecto, trabajando estrechamente con un cliente dedicado a la fabricación de acuarelas, con el objetivo de expandir su alcance al mundo digital y habilitar la venta en línea de sus productos. La iniciativa también incluyó el diseño y la implementación de una sección educativa en la página de inicio.

DAD JOKES

Esta mini aplicación utiliza una API para proporcionar chistes, permitiendo a los usuarios obtener chistes al azar o filtrarlos por palabra clave. Está desarrollada utilizando JavaScript puro, módulos y programación orientada a objetos. El uso de JavaScript puro demuestra un enfoque ligero y eficiente, aprovechando las capacidades del lenguaje para interactuar con la API y actualizar dinámicamente la interfaz de usuario.

GO TRAVEL

Go travel es un sitio de viajes que ofrece recomendaciones de destinos vacacionales, incluyendo sugerencias de hoteles y más. Se llevó a cabo bajo la metodología SCRUM. Además el sitio es totalmente responsive en cualquier dispositivo. Para la implementación, se utilizó JavaScript puro y se aplicó programación orientada a objetos.

RECOMMENDATIONS PLANTS

Es una pequeña aplicación que ofrece recomendaciones personalizadas de plantas según las respuestas proporcionadas en un formulario. Este proyecto se construyó utilizando JavaScript con el uso de módulos, LocalStorage para el almacenamiento local de datos, ESLint y Prettier para garantizar la consistencia del código, y se implementó un bundler con Webpack para gestionar las dependencias y optimizar el rendimiento de la aplicación.

SILVESTRE RESTAURANT

Este proyecto de diseño se centro en la remodelación del diseño del sitio web del Restaurante Silvestre en San José, con un enfoque específico en la mejora de la experiencia del usuario (UX/UI). La meta principal es optimizar la interacción de los usuarios con el sitio para asegurar una experiencia más fluida y atractiva.

TECHNOLOG

Este proyecto representa un ejercicio creativo y autónomo en el diseño de un sitio dedicado a la venta de aparatos electrónicos. Fue desarrollado íntegramente en Figma, con un enfoque centrado en la Experiencia de Usuario (UX/UI). La herramienta Figma se utilizó para plasmar visualmente todas las ideas y conceptos, permitiendo una colaboración efectiva y la creación de prototipos interactivos
